O que é o HTML?

(Te levo para cima.)

Hyper Text Markup é uma linguagem de marcação, utilizada para criar e estruturar o conteúdo de páginas web. Ele define a estrutura e o layout dos elementos presentes na página, como texto, imagens, links e vídeos. Permitindo a criação de páginas interativas e acessíveis.

Estrutura básica do HTML

(Te levo para cima.)

index.html:Utilizado para nomear a página principal do site.

tag:Toda tag deve possuir o sinal de menor "<", que irá "abrir ela, e o sinal de maior ">", que irá fechá-la.

  • Exemplo: "< html>< /html>".

Head: A "cabeça" do código. Nele teremos uma seção que contém informações que não serão exibidas diretamente na página, ele é apenas processado pelo navegador. É usado para incluir metadados, como o título da página, links para arquivos CSS, scripts da JavaScript, definição de caracteres etc.

  • Exemplo: "< link rel="stylesheet" href="estilos.css">".

Body: O "corpo" do código. É a seção principal de uma página web onde todo o conteúdo visível para para o usuário é inserido. Na estrutura, podemos ter parágrafos de textos, títulos, listas, tabelas, mídias, formulários etc.

  • Exemplo¹: "< p>Este é um parágrafo.< /p>".
  • Exemplo²:< img src="imagem.jpg" alt="Imagem de exemplo">

Exemplos de tags

(Te levo para cima.)

Tags: São elementos utilizados para marcar e estruturar o conteúdo da página web. Elas são representadas por meio de pares de sinais de menor "<" e maior ">" que envolvem o conteúdo que iremos marcar.

  • < i>< /i>: Deixa o texto itálico.
  • < strong>< /strong>: Destaca o texto.
  • < img src="" alt="">: Exibir uma imagem - < img src="endereço_da_imagem.jpg" alt="Texto alternativo.">
  • < hr>: Insere uma linha entre os conteúdos. Igual temos após a introdução (Hey, Dev.)
  • < h1>< /h1>: Definem o tamanho do texto, possuem níveis do h1 ao h6.
  • < p>< /p>: Basicamente, é o texto dentro de uma parágrafo.
  • < blockquote>< /blockquote>: Utilizado para fazer citações em um texto.

Atributos das tags

(Te levo para cima.)

Atributos: São usados para fornecer informações adicionais sobre uma tag e fornecem instruções para o navegador sobre como processar ou exibir o elemento.

  • id: Identifica exclusivamente um elemento na página.
      Declaração: < h1 id="myHeader">My Header< /h1>
  • class: Define a classe do elementos, permitindo que ele seja estilizado ou manipulado através do CSS ou JS. Diferente do id, ele é usado para agrupar elementos que possuem características semelhantes.
      Declaração: < div class="city">< /div>
  • src: Especifica o caminho para a origem de um recurso, como uma imagem ou um script.
      Declaração: < img src="url" alt="alternatetext">
  • href: Contém o destino de um link, seja para página interna ou externa.
      Declaração: < a href="https://www.w3schools.com">Visit W3Schools< /a>
  • alt: Fornece um texto alternativo para uma imagem, que é exibido caso ela não seja carregada.
      Declaração: < img src="img_chania.jpg" alt="Flowers in Chania">

Listas Ordenadas

(Te levo para cima.)

Ordered list: São utilizadas para criar listas com itens numerados ou com marcadores. Definidas pelo elementos < ol> e cada item da lista é marcado com o elemento < li> (listem item).

Declaração: < ol>< li>< /li>< /ol>

  1. Exemplo¹: Oi, eu sou o primeiro item da lista.
  2. Exemplo²: Oi, eu sou o segundo item da lista.

Listas Não Ordenadas

(Te levo para cima.)

Unordered list: São listas que não possuem uma ordem específica. Elas são definidas pelo elemento < ul>, e também possui o < li> em cada item da lista. Por padrão o marcados é um ponto •.

Declaração: < ul>< li>< /li>< /ul>

  • Exemplo¹: Oi, eu sou o primeiro item da lista não ordenada.
  • Exemplo²: Oi, eu sou o segundo item da lista não ordenada.

O que são os Links

(Te levo para cima.)

Links: Elementos utilizados para criar hiperlinks que permitem a navegação entre diferentes páginas ou recursos na web. Definidos pelo elemento ancor < a> e são usados para criar links para outras páginas, seções da mesma página, arquivos para download, endereços de e-mail etc.

Declaração: < a href="URL">Texto do link< /a>

  • Exemplo página externa: < a href="https://www.exemplo.com">Visite o site< /a>.
  • Exemplo seção na mesma página: < a href="#secao1">Ir para a Seção 1< /a>.

target + blank: O blank é um valor de atributo usado no elemento < a>. Quando o atributo target é definido com blank, ele faz com que o link seja aberto em uma nova guia do navegador.

Declaração: < a href="https://www.exemplo.com" target="_blank">Abrir em nova janela< /a>< /a>

title: O atributo title é usado para adicionar um balão de informações a um link. Quando o usuário passa o mouse sobre o link, o texto no atributo title é exibido em um pequeno balão de infos.

Declaração: < a href="#introducao" title="Clicando aqui, você irá para o início da página">(Te levo para cima.)< /a>
